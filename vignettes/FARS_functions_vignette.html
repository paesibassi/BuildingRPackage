FARS functions package Vignette

Federico Calore

2017-05-17

This package provides some convenient functions to extract, aggregate and visualise data from FARS - Fatality Analysis Reporting System. More information can be found at this link.

Low level data functions

The package includes some basic low-level functions to build a properly formatted filename and read a csv file. The data looks like the following:

tabl <- fars_read(datafile)

Data aggregations functions

Some convenient higher level functions are also available. These take as argument a vector of integers to determine the years. If a corresponding year is found in the data, the observations are aggregated by month and by year, and visualised as a contigency table, with Months in rows and Years in columns, as the following:

aggregated <- fars_summarize_years(c(2013, 2014, 2015), internal = TRUE)
aggregated

Plot functions

Finally, a plot function is also available. It takes two integer arguments, one for a USA State number and a second for the year. The resulting observations are then plotted on the selected State map, using the longitude and latitude information from the data file. Here’s a couple of examples:

fars_map_state(1, 2013, internal = TRUE)
fars_map_state(48, 2015, internal = TRUE)
